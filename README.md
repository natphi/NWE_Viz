# NWE Home Energy Dashboard

This repo contains a simple browser-based dashboard for viewing NorthWestern Energy hourly usage data.

## What It Does

The file `home-energy-dashboard.html` lets you:

- open the dashboard in a web browser
- drag in your own NorthWestern Energy CSV export
- explore charts for heatmaps, daily load shape, monthly totals, peak timing, and more

## How To Use It

1. Download your hourly usage CSV from NorthWestern Energy.
2. Open `home-energy-dashboard.html` in any modern web browser.
3. Drag your CSV file onto the page, or use the upload button.
4. Review the charts and use the filters to explore your usage patterns.

## Important Notes

- Your file is processed locally in your browser.
- The dashboard shows the most recent 8,760 hourly reads from the uploaded file.
- No install is required. It is just a single HTML file.

## Sharing It

You can also host this file with GitHub Pages so other people can open the page and upload their own CSV files.
